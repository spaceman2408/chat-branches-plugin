server plugin for [chat branches](https://github.com/spaceman2408/chat-branches)

turn on server plugins in `config.yaml`

```
enableServerPlugins: true
```

install the plugin by cloning this repository into the `plugins` folder or by downloading and extracting the zip file into the `plugins` folder

```
cd SillyTavern
cd plugins
git clone https://github.com/spaceman2408/chat-branches-plugin
```
